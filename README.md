# Reproductor de música — instrucciones

## Pasos para publicarlo con GitHub Pages

1. Ve a https://github.com y crea una cuenta si no tienes una.
2. Haz clic en el botón **+** (arriba a la derecha) → **New repository**.
3. Nombra el repositorio, por ejemplo `reproductor-musica`. Déjalo en **Public**. Crea el repositorio.
4. Dentro del repositorio, haz clic en **Add file → Upload files**.
5. Arrastra el archivo `index.html` de esta carpeta (sube solo ese archivo).
6. Haz clic en **Commit changes**.
7. Ve a la pestaña **Settings** del repositorio → menú lateral **Pages**.
8. En "Branch", selecciona `main` y la carpeta `/root`. Guarda.
9. Espera 1-2 minutos. GitHub te dará una URL como:
   `https://tu-usuario.github.io/reproductor-musica/`
10. Copia esa URL y pégala en Canva usando "Insertar link".

## Antes de subir

Abre `index.html` y reemplaza `URL_DE_TU_CANCION.mp3` por el link directo a tu archivo de audio
(alojado en Google Drive con acceso público, Dropbox, o cualquier servidor).

También puedes cambiar el texto `"Dale play a nuestra canción"` por el que prefieras.
